Курс мазмұны

HTML


-Visual Studio Code код редакторымен танысасыз
-Ол редакторда қалайша код жазуды 2 есе тездетуге болатынын білесіз
-Блок элементтерімен, мәтінмен, формалармен, сілтемелермен, сурет, видео, аудиомен, атрибуттармен жұмыс жасайсыз
-Сайттың дизайнны құрастыру, оны жақсарту, қателіктерді талдау жайлы арнайы жеке сабақ аласыз

CSS


-Селектор, элемент моделі, flexbox жүйесін үйренесіз
-Мәтінді стильдейсіз, блокты әсемдейсіз
-display, position, transform сипаттамаларына толық түсінік аласыз
-Жайттармен жұмыс жасайсыз
-Анимацияны үйренесіз
-px, %, em, rem, ch, vw, wh өлшемдерімен танысасыз
-responsive design(сайттың барлық құрылғыда дұрыс көрінуі) техникасын үйренесіз
-медиа-сұраныс (медиа-запрос) не екенін білесіз
-иконкалармен жұмыс істейсіз

Javascript


-Элементтерді таңдауды, оларға функция қолдануды үйренесіз
-Элементтің классына әсер етуді білесіз
-Формалармен жұмыс істейсіз
-Regex тілі арқылы жазылған email, құпиясөзді тексеруді үйренесіз
-Тексерістің нәтижесіне байланысты ақпаратты өзгерте алатындай боласыз
-Элементке байланысты жайттарға қатысты функция орындайсыз
